🌐 Advanced Web Scraper (Streamlit App)

A simple and interactive E-Commerce Web Scraper built using Streamlit, Requests, and BeautifulSoup.

This app scrapes product details from the demo website Books to Scrape and allows users to export the data as a CSV file.

🚀 Features

📄 Scrape multiple pages (1–5 pages)

📦 Extract product details:

Product Name

Price

Rating

Page Number

📊 Display scraped data in an interactive table

📥 Download results as CSV

⚡ Real-time progress bar during scraping

🛠️ Tech Stack

🐍 Python

🎈 Streamlit

🌍 Requests

🍲 BeautifulSoup

📊 Pandas
